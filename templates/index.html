Train Ticket Price Prediction

The Train Ticket Price Prediction project leverages Machine Learning to accurately estimate railway ticket prices based on various travel parameters such as source, destination, train type, journey time, and class. The model is built using Random Forest Regression, which captures non-linear relationships between features for high prediction accuracy.

The system performs data preprocessing, feature engineering, and hyperparameter tuning to improve model performance. It helps passengers or railway systems forecast ticket prices dynamically and supports data-driven decision-making in fare management.

Text To Title

The Text To Title project is a lightweight web application built using Flask and Selenium that automatically extracts and displays the title of any web page from its URL.

Users simply input a website link, and the system uses Selenium’s browser automation to fetch the HTML metadata, parse the document, and return the page title in real time.

Designed with simplicity and efficiency in mind, this project demonstrates seamless integration of Python web frameworks and browser automation tools for quick web content extraction.

Python Flask Selenium Web Development
